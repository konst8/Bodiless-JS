# `@bodiless/core-ui`

Default UI for bodiless core admin components.

## Usage

```javascript
import { PageEditor, ContextWrapper } from '@bodiless/core-ui';
export default () => (
  <PageEditor>
    <ContextWrapper>
      <MyComponent />
    </ContextWrapper>
  </PageEditor>
);
```
